# Brownie Boo — Website UI Kit

## Overview
High-fidelity interactive recreation of the Brownie Boo landing page and order flow.

## Pages Covered
| Screen | Description |
|---|---|
| **Home** | Hero + Flavours + Menu sections |
| **Choose** | Box type selection (3 options) |
| **Order** | Flavour picker + live cart |

## Components
- `Header.jsx` — Sticky nav with logo, links, CTA
- `Hero.jsx` — Hero section with headline, stats, CTA buttons
- `FlavourGrid.jsx` — 6-flavour card grid with popular badges
- `MenuCards.jsx` — 3 box-type cards (Classic, Party, Special)
- `OrderFlow.jsx` — Flavour quantity picker + cart sidebar
- `Footer.jsx` — Dark footer with links and contact info

## Usage
Open `index.html` for the full interactive prototype.

## Notes
- Font substitution: `cursive` → Playfair Display (Google Fonts)
- Icons: Font Awesome 6.0.0 via CDN
- Images referenced from `../../assets/`
